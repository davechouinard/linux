# dotfiles

Configurations for my core set of tools:

* i3 - Window Manager
    * i3blocks
* termite - Terminal
* tmux - Terminal multiplexer
* vim - Editor
    * airline
    * ctrlp
    * gitgutter
    * nerdtree
    * tagbar
    * ultisnips
* zsh - Shell
    * autosuggestions
    * completions
    * syntax-highlighting
    * powerlevel9k

![Screenshot](dotfiles.png)

## Usage

Install everything, plugins etc. on a new machine:

`./full-setup.sh`

Dotfiles only:

`./setup.sh`
